# Luo Coding Data

A comprehensive, curated coding dataset for AI training and development.

## Contents

### Programming Languages
- Python, JavaScript, Java, C/C++, Go, Rust, TypeScript
- C#, PHP, Ruby, Swift, Kotlin, Dart, Scala
- R, Perl, Lua, Haskell, Julia, Zig

### Cybersecurity
- CVEs and exploits
- Security tools and frameworks
- Payloads and wordlists

### Web Frameworks
- React, Vue, Angular, Next.js
- Django, Flask, Express, Spring, Laravel

### Mobile Development
- Android, iOS, Flutter, React Native

### Databases
- MySQL, PostgreSQL, MongoDB examples

### Machine Learning
- Scikit-learn, TensorFlow, PyTorch examples

### DevOps
- Docker, Kubernetes, Terraform, CI/CD

### Algorithms
- Sorting, searching, graph algorithms
- Data structures implementations

## Usage

```bash
# Clone the dataset
git clone https://github.com/luokai0/luo-data

# Navigate to the dataset
cd luo-data/luo_coding_data
```
