# luo_coding_data - Comprehensive Coding Dataset

## Structure

```
luo_coding_data/
├── python_core/         # Python standard library & core packages
├── python_frameworks/   # Django, Flask
├── js_core/            # Node.js core modules
├── js_frameworks/      # React, Vue, Express
├── java_spring/        # Spring Framework
├── ruby_rails/         # Ruby on Rails
├── ml_frameworks/      # Keras, scikit-learn
├── cybersec/           # Cybersecurity tools & exploits
│   ├── exploits/       # Metasploit, SQLMap, etc.
│   ├── web_attacks/   # Web vulnerability attacks
│   ├── network/       # Network security tools
│   ├── wordlists/     # Password wordlists
│   └── analysis/      # Security analysis scripts
├── databases/          # PostgreSQL, Redis, MongoDB
├── devops/             # Docker, Kubernetes, Ansible, Terraform
├── algorithms/         # Sorting, searching, graph algorithms
└── c_cpp/             # C/C++ code samples
```
